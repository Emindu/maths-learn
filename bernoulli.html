Discrete Distribution

Bernoulli Distribution

The simplest probability distribution — a single experiment with exactly two outcomes: success or failure.

Real-World Use Cases

Wherever there's a single yes/no question, the Bernoulli distribution is the model.

🪙

Coin Flip

A fair coin lands heads (success) with p = 0.5. Each individual flip is one Bernoulli trial.

📊

A/B Testing

A single visitor either clicks a button (success) or doesn't. Conversion rate p is estimated from many Bernoulli observations.

🏭

Quality Control

A manufactured part is either defective (failure) or not. Each inspection is one Bernoulli trial with defect probability p.

💊

Clinical Trials

A patient either responds to a drug (success) or doesn't. Response probability p is the key parameter of interest.

Key Properties at a Glance

  • Only two possible outcomes: X = 0 (failure) or X = 1 (success).
  • When p = 0.5 the distribution is perfectly symmetric — both outcomes equally likely.
  • When p > 0.5 the distribution is left-skewed (success is more likely).
  • When p < 0.5 the distribution is right-skewed (failure is more likely).
  • Variance peaks at p = 0.5 and drops to 0 as p → 0 or p → 1 (certainty reduces uncertainty).
  • The Bernoulli is the building block of most other discrete distributions.

Connections to Other Distributions

→ Binomial Sum of n independent Bernoulli(p) trials is Binomial(n, p). Bernoulli is the special case n = 1.
→ Geometric The number of Bernoulli trials until the first success follows a Geometric(p) distribution.
→ Negative Binomial Counting trials until r successes generalises the Geometric using the same Bernoulli building block.
→ Beta (conjugate prior) When estimating p from data, the Beta distribution is the conjugate prior for the Bernoulli likelihood.

Probability Mass Function (PMF)

A random variable X follows a Bernoulli distribution with parameter p ∈ [0, 1] if it takes only the values 0 and 1:

\[ P(X = k) = p^k (1-p)^{1-k}, \quad k \in \{0, 1\} \]

Equivalently written as:

\[ P(X = k) = \begin{cases} 1-p & \text{if } k = 0 \\ p & \text{if } k = 1 \end{cases} \]

Effect of the Parameter p

p = 0: X is always 0. The distribution has zero variance — there is no randomness.

p = 0.5: The distribution is perfectly symmetric. Both outcomes are equally likely and variance is at its maximum value of 0.25.

p = 1: X is always 1. Again zero variance — no randomness.

As p increases from 0 to 0.5, variance increases (more uncertainty). As p increases from 0.5 to 1, variance decreases (less uncertainty). This makes intuitive sense: when an outcome is nearly certain, there is little surprise.

Derivation of Mean and Variance

Mean (Expected Value):

\[ E[X] = 0 \cdot (1-p) + 1 \cdot p = p \]

Variance: Using \( \text{Var}(X) = E[X^2] - (E[X])^2 \):

\[ E[X^2] = 0^2(1-p) + 1^2 p = p \] \[ \text{Var}(X) = p - p^2 = p(1-p) \]

Note: since X only takes values 0 and 1, we have X² = X, so E[X²] = E[X] = p.
